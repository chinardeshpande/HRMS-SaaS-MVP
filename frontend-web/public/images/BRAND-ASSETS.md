## Logo Usage Guidelines

### ✅ Do
- Use on white/light backgrounds primarily
- Use on dark backgrounds (it works beautifully)
- Maintain clear space equal to icon width around logo
- Scale proportionally
- Use provided SVG files (vector, infinite scaling)
- Use the mark (aurora-mark.svg) for favicons and tiny spaces

### ❌ Don't
- Change colors or gradients
- Rotate, skew, or distort
- Add effects (shadows, outlines, glows)
- Place on busy backgrounds
- Separate icon from text arbitrarily
- Recreate or redraw the logo

## File Formats

All assets are provided as **SVG** (Scalable Vector Graphics):
- ✅ Infinite scaling without quality loss
- ✅ Small file size
- ✅ Works everywhere (web, print, mobile)

For raster needs (PNG, JPG), export from SVG at 2x–3x final size.

## Design Philosophy

**The Rising Arc**
- Represents dawn — the moment light breaks through darkness
- Symbolizes growth, potential, awakening
- Minimal geometry = timeless, iconic
- Like Nike's swoosh or Apple's apple — instantly recognizable

**The Person**
- Represents humanity at the core of HR
- Shows individual growth within the broader arc
- Optional (use the mark without person for ultra-minimal)

**The Gradient**
- Dark to light = journey from challenge to success
- Blue palette = trust, professionalism, stability
- Subtle gradient = modern, digital, forward-looking
