# Feature Images Directory

This directory contains illustrations for each HRMS module feature.

## Required Images

Please save the following images in this directory:

1. **employee-management.svg** - Employee management illustration
2. **onboarding.svg** - Onboarding process illustration
3. **attendance.svg** - Time & attendance tracking illustration
4. **leave.svg** - Leave management illustration
5. **performance.svg** - Performance review illustration
6. **exit.svg** - Exit management illustration

## Image Specifications

- **Format**: SVG, PNG, or JPG
- **Recommended Size**: 800x600px or similar aspect ratio
- **Style**: Consistent illustration style with primary blue color scheme
- **File Size**: Optimize for web (< 200KB per image)

## Usage

These images are automatically loaded in:
- Landing page feature cards
- Feature detail pages
- Use case sections

## Fallback

If images are not available, the system will show gradient backgrounds with icons as placeholders.
